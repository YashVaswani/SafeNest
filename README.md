# SafeNest 🛡️

SafeNest is a privacy-first trust and verification platform designed for gated residential societies. The system helps RWAs, residents, and security guards manage and verify household helpers such as maids, drivers, cooks, and caretakers using QR-code-based identity verification.
Unlike traditional security systems, SafeNest does not use Aadhaar, biometrics, facial recognition, or surveillance tracking. The platform focuses on accountability, safety, and community trust while maintaining user privacy.

## 🚀 Features

- QR-based helper verification
- Role-based login system
- Admin dashboard for RWAs
- Resident helper directory
- Security guard QR scanner
- Entry/Exit logging
- Complaint & feedback system
- Helper work history management
- Responsive dashboard UI

## 🛠️ Tech Stack

- Python
- Flask
- SQLite
- SQLAlchemy
- Flask-Login
- Bootstrap 5
- HTML/CSS/JavaScript
- QRCode Library


**⚙️ Installation**
1. Clone Repository
git clone https://github.com/your-username/safenest.git
cd safenest
2. Create Virtual Environment
python -m venv venv
3. Activate Virtual Environment
Windows
venv\Scripts\activate
Mac/Linux
source venv/bin/activate
4. Install Dependencies
pip install -r requirements.txt
5. Run Application
python app.py

Admin (RWA)
Manage helpers

Monitor logs

View work history
Submit feedback
Security Guard
Scan QR codes
Verify helper status
Maintain entry logs
